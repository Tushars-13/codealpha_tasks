# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project aims to classify Iris flowers into three different species using Machine Learning algorithms.

The Iris dataset contains measurements of flower characteristics such as sepal length, sepal width, petal length, and petal width. Using these features, machine learning models are trained to predict the species of a flower.

### Species Classified

* Iris-setosa
* Iris-versicolor
* Iris-virginica

---

## 🎯 Objective

The objective of this project is to:

* Understand the machine learning workflow.
* Perform Exploratory Data Analysis (EDA).
* Visualize feature relationships.
* Train classification models.
* Evaluate model performance.
* Predict flower species based on measurements.

---

## 📂 Dataset Information

Dataset Features:

* SepalLengthCm
* SepalWidthCm
* PetalLengthCm
* PetalWidthCm

Target Variable:

* Species

Dataset Size:

* 150 Samples
* 3 Classes
* Balanced Dataset (50 samples per class)

---

## 🔍 Exploratory Data Analysis (EDA)

The following analyses were performed:

* Dataset inspection

### Key Findings

* No missing values were found.
